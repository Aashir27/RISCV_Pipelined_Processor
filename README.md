# RISC-V Pipelined Processor

## Overview
A Pipelined RISC-V Processor designed as part of a Computer Architecture project. The repository incorporates structural and behavioral design tasks illustrating processor data paths, control unit operations, stalling, and hazard handling.

## Contents
- **Tasks (1-3)**: Iterative design files detailing the processor's datapath enhancements and pipeline registers.
- **CA_Project_Report.pdf**: Comprehensive documentation of the architecture, design choices, state diagrams, and pipeline logic.
